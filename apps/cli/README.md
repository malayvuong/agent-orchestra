# @malayvuong/agent-orchestra

Structured multi-agent review for plans and code, packaged as a CLI.

## Install

```bash
npm install -g @malayvuong/agent-orchestra
```

Requires Node.js `>= 20`.

## Quick Start

```bash
agent-orchestra init
agent-orchestra run --target ./docs/plan.md --superpower plan-review --auto-apply
agent-orchestra serve --mcp
```
